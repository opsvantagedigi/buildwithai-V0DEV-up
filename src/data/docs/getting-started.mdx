# Getting started

This guide walks you through creating your first site using BUILD WITH AI.

## 1. Create a project

1. Sign up or sign in to the platform.
2. Click "Create New Site" and enter your business name.

```bash
# example: initialize via CLI (optional)
npx buildwithai init my-site
```

## 2. Pick a template

Choose a template category and let the AI generate pages for you.

## 3. Customize and publish

Fine-tune content, connect a domain, and publish.
