Build With AI — 12-month Roadmap (Q1 → Q4)
=========================================

Q1 — Foundations (0–3 months)
- Repo & CI/CD baseline
- Identity (email verification + JWT)
- Minimal AI Website Builder (static generator + Vercel deploy)
- Mailcow PoC for transactional email
- Operator Payments skeleton + DB schema

Q2 — Payments & CRM (3–6 months)
- Full Operator Payments integration (NOWPayments primary)
- Checkout and webhook flows
- CRM events logging + operator UI
- Billing & subscription basics

Q3 — Automation & Fraud (6–9 months)
- Deterministic fraud-risk scoring & escalation
- MARZ Orchestrator automation for digests/escalations
- AI-generated operator summaries
- Live chat MVP (MARZ)

Q4 — Multi-tenant & Live Agent (9–12 months)
- Multi-tenant support and tenant scoping
- Advanced dashboards and role controls
- Live video agent (MARZ) MVP
- Governance, docs, and hardening

Each quarter broken into milestones (see `docs/MILESTONES.md`).
